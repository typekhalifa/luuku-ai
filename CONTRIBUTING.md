# Contributing

Thank you for your interest in contributing to Luuku AI.

## Development Philosophy

- Keep modules small and focused.
- Prefer shared services over duplicated logic.
- Agents should never access storage directly.
- Maintain clear separation between agents, services, and storage.

## Workflow

1. Create a feature branch.
2. Make focused commits.
3. Run tests.
4. Open a Pull Request.

More contributor guidelines will be added as the project grows.
